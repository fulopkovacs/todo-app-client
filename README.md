# Todo App Client

> ⚠️ Most of the work on this repo was blindly done via coding agents. ^^ (The original trytanstackdb.com repo is not affected by this issue.)

A todo app using [TanStack DB](https://tanstack.com/db/latest) and [Query](https://tanstack.com/query/latest), extracted from [fulopkovacs/trytanstackdb.com](https://github.com/fulopkovacs/trytanstackdb.com), for a YouTube video.

This is not a proper reference app!

In the YouTube video, I take this app and add real-time sync to it with Electric. Check out the [`use-electric` branch](https://github.com/fulopkovacs/todo-app-client/tree/use-electric) for the end result.
